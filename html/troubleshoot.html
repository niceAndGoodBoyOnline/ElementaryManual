Troubleshooting Linux



Since elementary OS is based on Ubuntu 18.04, using your favourite search engine with the keyword "Ubuntu" paired with the issue you are having will return other users solutions.

Some common issues that occur while installing and configuring elementary OS:

  • If your bootable USB Drive doesn't show up in the boot menu:

    • Try using a different USB port.
    • You might have to turn on USB boot:

        • NOTE: Because different computers have their own BIOS Setup Utilities, refer to the bottom of the BIOS screen to look for what a specific key does.

      1. Turn your computer off.
      2. Turn your computer on.
      3. Press the biossetup key on your computer's keyboard.
      4. Select the Boot tab using the arrow keys on the keyboard.
      5. Move USB to be first in the boot sequence using the key map at the bottom of the screen.
      6. Save the change and exit the BIOS setup.
      7. The computer will restart now using the new settings, booting from the USB drive.

  • If you get an error in the terminal due to broken packages:

    1. Open your terminal. (Click here if you don't know how to.)
    2. Type "sudo apt clean".
    3. Type "sudo apt install -f".
    4. Type "sudo dpkg --configure -a".
    5. Type "sudo apt update".
